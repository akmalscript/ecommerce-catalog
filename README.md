# ecommerce-catalog

A Vue.js web application for displaying product catalog from an e-commerce API. The app shows men's and women's clothing products with dynamic theming based on product category.

## Output

Check out [The Live Page](https://ecommerce-catalog-akmalscript.vercel.app/)

## Features

- Display product details from [Fake Store API](https://fakestoreapi.com)
- Dynamic theme colors based on product category (blue for men's, pink for women's)
- Unavailable product page for non-clothing categories
- Skeleton loading animation
- Responsive design

## Technologies

The technologies used to build this project:

- Vue.js 3
- Vite
- CSS3
- Fake Store API

## Getting Started

1. Clone the repository:
```
git clone https://github.com/akmalscript/ecommerce-catalog.git
cd ecommerce-catalog
```

2. Install dependencies:
```
npm install
```

### Run Development Server
```
npm run dev
```

### Build for Production
```
npm run build
```

### Preview Production Build
```
npm run preview
```
